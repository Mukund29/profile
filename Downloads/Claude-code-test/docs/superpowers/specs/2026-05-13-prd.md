## 1. Product Vision & Goals

### 1.1 Vision Statement
> **"Your financial co-pilot — always tracking, always honest, never judgmental."**

FinTrack is the first cross-platform mobile personal finance app that auto-captures transactions across India, the United States, and the United Kingdom without manual data entry — and enforces financial discipline through weekly scoring, all at an accessible $2.99/month price point.

### 1.2 Product Goals

| Goal | Metric | Target |
|------|--------|--------|
| Eliminate manual transaction entry | % transactions auto-captured | ≥ 60% |
| Deliver financial discipline accountability | Weekly discipline score adoption | ≥ 70% of active users |
| Reduce onboarding friction | Time-to-dashboard | < 45 seconds |
| Maximise trial-to-paid conversion | Conversion rate | ≥ 35% |
| Retain users long-term | Day-30 retention | ≥ 20% |

### 1.3 Design Principles

1. **Zero-entry by default** — every auto-captured transaction saves the user 15 seconds
2. **Progressive disclosure** — don't ask for anything upfront that isn't needed immediately
3. **Honest feedback** — discipline score is factual, not gamified; it doesn't lie for engagement
4. **Privacy by design** — on-device SMS parsing; E2E encryption; zero bank credentials stored
5. **Speed over features** — a P95 screen load of < 1.5s matters more than a sixth report type

| FR-C12 | User notified of new auto-parsed transaction via push notification; notification tap deep-links to review screen | S | DA |

**Acceptance Criteria — FR-C02 (On-Device SMS Privacy):**
- Given the Android app reads an SMS from HDFC Bank containing a debit transaction
- When the SMS is processed
- Then no raw SMS text is transmitted to any server; only `{amount, merchant, date, currency, type}` is sent

**Acceptance Criteria — FR-C11 (Parse Accuracy):**
- Given a test corpus of 500 known-good bank SMS messages across 5 supported banks
- When parsed by the on-device parser
- Then ≥ 460 (92%) produce correctly extracted transactions with all required fields

**Acceptance Criteria — FR-E10 (Discipline Score Formula):**
- Given a user with 50/30/20 targets spent 60% Needs / 35% Wants / 5% Savings
- When the weekly summary is computed
- Then discipline score = 100 - ((|60-50|×0.5) + (|35-30|×0.3) + (|5-20|×0.2)) × 2 = 100 - (5 + 1.5 + 3) × 2 = 81
